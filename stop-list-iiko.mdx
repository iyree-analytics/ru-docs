## Получение API логина iikoweb

Зайдите в iikoweb (\<ваш уникальный поддомен\>.iikoweb.ru) и авторизуйтесь.

1. Зайдите в раздел **_Внешние заказы_** → **_Настройка Cloud API_**

<Frame>
  ![Iikoweb – API Token – 1](/images/integrations/iiko/iikoweb-–-API-token-–-1.png)
</Frame>

2. На вкладке **_ИНТЕГРАЦИИ_** нажмите кнопку **_Добавить интеграцию_**

<Frame>
  ![Iikoweb – API Token – 2](/images/integrations/iiko/iikoweb-–-API-token-–-2.png)
</Frame>

3. На вкладке **_ОСНОВНОЕ_** заполните следующие поля:

- **_Имя API логина_**. Укажите любое понятное вам название, идентифицирующее нас как интеграцию, например, _IYREE_
- **_Email_**. Укажите [_info@iyree.ru_](mailto:info@iyree.ru)
- **_Шаблон прав_**. Выберите **_Все права_**

<Frame>
  ![Iikoweb – API Token – 3](/images/integrations/iiko/iikoweb-–-API-token-–-3.png)
</Frame>

4. Перейдите на вкладку **_ПОДКЛЮЧЕННЫЕ ТОЧКИ_** и нажмите кнопку **_Добавить точку._**

<Frame>
  ![Iikoweb – API Token – 4](/images/integrations/iiko/iikoweb-–-API-token-–-4.png)
</Frame>

5. В открывшемся списке выберите точки, к которым хотите дать доступ нашей интеграции. Если хотите дать доступ ко всем точкам, нажмите галочку слева от первой строки **_Название точки_**, чтобы выбрать все точки. Нажмите кнопку **_Добавить_** снизу экрана.

<Frame>
  <img
    src="/images/integrations/iiko/iikoweb-–-API-token-–-5.png"
    alt="Iikoweb – API Token – 5"
    title="Iikoweb – API Token – 5"
  />
</Frame>

6. Перейдите обратно на вкладку **_ОСНОВНОЕ_** и скопируйте API ключ.

<Frame>
  ![Iikoweb – API Token – 6](/images/integrations/iiko/iikoweb-–-API-token-–-6.png)
</Frame>

7. Передайте API ключ вашему менеджеру IYREE.
