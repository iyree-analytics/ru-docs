## Быстрый старт

### Формат конфигурации

Конфигурация графиков задаётся в формате JSON или YAML. Каждый график имеет обязательное поле `type`, определяющее тип визуализации.

### Пример простого графика

```
type: tile
main_measure: sales.total_revenue
```

### Пример таблицы

```
type: table_v2
columns:
  - type: dimension
    member:
      kind: basic
      value: products.name
    title: Товар
  - type: measure
    member: sales.total_revenue
    title: Выручка
```

---

## Общие поля для всех графиков

| Поле                     | Тип         | Обязательное | Описание                                 |
| :----------------------- | :---------- | :----------- | :--------------------------------------- |
| `type`                   | `ChartType` | ✅            | Тип графика                              |
| `compare_period_enabled` | `boolean`   | ❌            | Включить сравнение с предыдущим периодом |

## Поддерживаемые типы графиков

| Тип                | Идентификатор    | Описание                         |
| :----------------- | :--------------- | :------------------------------- |
| Плитка             | `tile`           | Одиночный показатель             |
| Мульти-плитка      | `multi_tile`     | Несколько показателей            |
| Таблица v1         | `table`          | Простая таблица (устаревшая)     |
| Таблица v2         | `table_v2`       | Расширенная таблица              |
| Сводная таблица v1 | `pivot_table`    | Простая сводная (устаревшая)     |
| Сводная таблица v2 | `pivot_table_v2` | Расширенная сводная таблица      |
| Столбчатая v1      | `bar`            | Простая столбчатая (устаревшая)  |
| Столбчатая v2      | `bar_v2`         | Расширенная столбчатая           |
| Донат              | `donut`          | Кольцевая диаграмма (устаревшая) |
| Круговая           | `pie`            | Круговая/кольцевая диаграмма     |
| Динамическая       | `dynamic_chart`  | Кастомные ECharts                |

> ⚠️ Версии v1 графиков (`table`, `pivot_table`, `bar`, `donut`) считаются устаревшими. Рекомендуется использовать версии v2 с расширенной функциональностью.
